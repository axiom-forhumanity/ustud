# Letters A to M

English uses 26 letters. Let's learn the first 13: A, B, C, D, E, F, G, H, I, J, K, L, M.

## The Letters

- **A** (ay) - apple, ant
- **B** (bee) - ball, book
- **C** (see) - cat, cup
- **D** (dee) - dog, door
- **E** (ee) - egg, elephant
- **F** (eff) - fish, flower
- **G** (jee) - girl, goat
- **H** (aych) - house, hand
- **I** (eye) - ice, insect
- **J** (jay) - jump, juice
- **K** (kay) - kite, key
- **L** (ell) - lion, leaf
- **M** (em) - moon, mother

## Practice

Say each letter out loud. Find things around you that start with each letter.
