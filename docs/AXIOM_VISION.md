# AXIOM Vision — UStud

## What we are building
UStud is an offline-first AI learning platform designed for learners who lack reliable internet access. It runs on modest hardware (4–8 GB RAM laptops) and provides curriculum-guided education with a local AI tutor.

## Core values
- **Accessibility** — Works offline; designed for low-resource environments
- **Modularity** — Courses are plugins; easy to add new subjects
- **Dignity** — Respectful, supportive learning experience
- **Quality** — Solid foundation before feature expansion

## Product pillars
1. Curriculum-guided learning (basics to advanced within each course)
2. Offline AI tutor (local LLM via Ollama)
3. Language support (translation to Dari/Persian for learners)
4. Content library (Wikipedia ZIM + curated course packs + PDFs)
5. Modular courses (Math, Science, English, Mental Health, Fitness & Nutrition)

## Success looks like
- A learner can open UStud on a cheap laptop, pick a course, and learn without internet
- The platform is stable, documented, and extensible
- New modules and content can be added without rewriting core systems

## What employees should optimize for
- Stability and clarity over flashy features
- Offline reliability
- Clean documentation and progress logging
- Incremental improvements aligned with this vision
